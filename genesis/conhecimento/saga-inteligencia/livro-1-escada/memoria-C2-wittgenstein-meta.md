## 5. Elementos Biográficos Essenciais

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   WITTGENSTEIN — VIDA (para a narrativa)                                    │
│                                                                             │
│   • Nasceu em Viena, 1889. Família mais rica da Áustria.                    │
│   • Três irmãos se suicidaram. Ele considerou muitas vezes.                 │
│   • Engenheiro de formação. Foi estudar lógica com Frege e Russell.         │
│   • Escreveu o Tractatus nas trincheiras da Primeira Guerra.                │
│   • Achou que tinha resolvido todos os problemas da filosofia.              │
│   • Abandonou a filosofia. Virou professor primário, jardineiro, monge.     │
│   • Voltou. Destruiu o Tractatus com as Investigações.                      │
│   • Morreu em 1951 dizendo: "Diga a eles que tive uma vida maravilhosa."    │
│                                                                             │
│   PARALELO COM GENESIS:                                                     │
│   • Wittgenstein construiu sistema → achou completo → destruiu              │
│   • GENESIS constrói sistema → achará completo? → destruirá?                │
│   • "A escada deve ser jogada fora depois de subir"                         │
│                                                                             │
```
