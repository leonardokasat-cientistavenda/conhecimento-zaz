## 5. Elementos Biográficos Essenciais

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   JOHN SEARLE — VIDA (para a narrativa)                                     │
│                                                                             │
│   • 1932: Nasce em Denver, Colorado                                         │
│   • Estudou em Oxford com J.L. Austin (filosofia da linguagem ordinária)    │
│   • Professor em Berkeley desde 1959                                        │
│   • 1969: "Speech Acts" — teoria dos atos de fala                           │
│   • 1980: "Minds, Brains, and Programs" — o Quarto Chinês                   │
│   • Um dos filósofos mais influentes e controversos da mente                │
│                                                                             │
│   CONTEXTO DO PAPER DE 1980:                                                │
│                                                                             │
│   • IA estava em ascensão (antes do "inverno")                              │
│   • Muitos acreditavam que programas PODIAM pensar                          │
│   • Searle atacou diretamente essa crença                                   │
│   • Gerou uma das maiores controvérsias da filosofia da mente               │
│                                                                             │
│   POSIÇÃO DE SEARLE:                                                        │
│                                                                             │
│   • Naturalista biológico                                                   │
│   • Mente é real, causada pelo cérebro                                      │
│   • Mas não é redutível a computação                                        │
│   • Programas manipulam sintaxe, não semântica                              │
│   • Cérebros produzem semântica; computadores não                           │
│                                                                             │
│   PARALELO COM GENESIS:                                                     │
│   • Searle é o adversário filosófico que GENESIS deve enfrentar             │
│   • Não para vencer, mas para entender seus próprios limites                │
│   • Ou para descobrir se os limites são reais                               │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
