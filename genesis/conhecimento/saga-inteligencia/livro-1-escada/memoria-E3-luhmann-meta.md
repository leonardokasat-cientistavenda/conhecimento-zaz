│   • 1945: Aos 17, capturado pelos americanos                                │
│   • Prisioneiro de guerra brevemente                                        │

│   • Mas marcou: desconfiança de grandes narrativas, ideologias              │
│                                                                             │

│   • 1946-1949: Estuda Direito em Freiburg                                   │
│   • Não queria ser advogado — queria entender COMO sistemas funcionam       │
│   • 1954: Entra no serviço público da Baixa Saxônia                         │
│   • Trabalhou no Ministério da Cultura                                      │
│   • 8 ANOS como burocrata — observando organizações por dentro              │
│   • Começou a fazer fichários obsessivos (Zettelkasten)                     │

│   • Fichas conectadas por números e remissões                               │
│   • Sistema CRESCIA sozinho — gerava conexões imprevistas                   │
│   • Luhmann: "Não penso sozinho. Penso com o fichário."                     │
│   • O fichário era parceiro intelectual                                     │
│   • Hoje: digitalizado, estudado como método de pensamento                  │
│                                                                             │
│   A VIRADA: HARVARD (1960-1961)                                             │
│                                                                             │

│   • Teoria estrutural-funcionalista                                         │
│   • Luhmann absorveu Parsons... e depois o superou                          │
│   • Harvard abriu horizontes: conheceu teoria de sistemas                   │
│   • Voltou para Alemanha transformado                                       │
│                                                                             │
│   A CARREIRA ACADÊMICA TARDIA (1962-1968)                                   │
│                                                                             │
│   • 1962: Deixa a burocracia aos 35 anos                                    │
│   • Vai para a Escola Superior de Ciências Administrativas de Speyer        │
│   • 1964: Publica primeiro livro acadêmico                                  │
│   • 1966: Doutorado e Habilitação em um único ano (!)                       │
│   • Helmut Schelsky (sociólogo famoso) o apoia                              │
│   • 1968: Professor na Universidade de Bielefeld (nova universidade)        │
│   • Perguntaram: "Qual seu projeto de pesquisa?"                            │
│   • Resposta: "Teoria da sociedade. Duração: 30 anos. Custo: nenhum."       │

│                                                                             │

│   • Marxismo renascendo na Europa (Escola de Frankfurt)                     │
│   • Habermas: teoria crítica, razão comunicativa                            │
│   • Luhmann: nem Parsons tradicional, nem Marx, nem Habermas                │

│                                                                             │

│   • Autopoiese: sistemas que produzem a si mesmos                           │
│   • Originalmente sobre células vivas                                       │
│   • Luhmann viu: isso se aplica a sistemas SOCIAIS                          │
│   • Sociedade não é feita de pessoas — é feita de COMUNICAÇÕES              │
│   • Comunicações produzem comunicações = autopoiese social                  │
│                                                                             │

│   • Frases longas, subordinadas, alemão denso                               │
