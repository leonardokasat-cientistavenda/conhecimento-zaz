## 5. Contexto Histórico-Intelectual

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   O MUNDO DA CRISE DOS FUNDAMENTOS                                          │
│                                                                             │
│   ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│   A CRISE (1900-1930):                                                      │
│                                                                             │
│   • 1901: Paradoxo de Russell — conjunto de todos os conjuntos              │
│   • "O conjunto dos conjuntos que não contêm a si mesmos"                   │
│   • Contém a si mesmo? Paradoxo lógico.                                     │
│   • A matemática tinha contradições em sua base!                            │
│                                                                             │
│   ─────────────────────────────────────────────────────────────────────     │
│                                                                             │
│   AS TRÊS RESPOSTAS:                                                        │
│                                                                             │
│   1. LOGICISMO (Russell, Whitehead):                                        │
│      Reduzir matemática à lógica pura.                                      │
│      "Principia Mathematica" — 2000 páginas para provar 1+1=2               │
│                                                                             │
│   2. INTUICIONISMO (Brouwer):                                               │
│      Rejeitar partes da matemática (terceiro excluído)                      │
│      Só aceitar o que pode ser construído mentalmente                       │
│                                                                             │
│   3. FORMALISMO (Hilbert):                                                  │
│      Axiomatizar tudo. Provar consistência dos axiomas.                     │
│      Matemática como jogo de símbolos com regras seguras.                   │
│                                                                             │
│   ─────────────────────────────────────────────────────────────────────     │
│                                                                             │
│   O PROGRAMA DE HILBERT:                                                    │
│                                                                             │
│   David Hilbert propôs:                                                     │
│   1. Formalizar toda matemática em sistema de axiomas                       │
│   2. Provar que o sistema é CONSISTENTE (sem contradições)                  │
│   3. Provar que o sistema é COMPLETO (toda verdade é provável)              │
│   4. Provar que há PROCEDIMENTO para decidir qualquer proposição            │
│                                                                             │
│   Se conseguisse: matemática seria SEGURA, DECIDÍVEL, COMPLETA.             │
│                                                                             │
│   ─────────────────────────────────────────────────────────────────────     │
│                                                                             │
│   GÖDEL DESTRUIU O SONHO (1931):                                            │
│                                                                             │
│   Primeiro Teorema:                                                         │
│   Em qualquer sistema formal consistente e suficientemente rico             │
│   para expressar aritmética, existem proposições verdadeiras                │
│   que NÃO PODEM ser provadas dentro do sistema.                             │
│                                                                             │
│   Segundo Teorema:                                                          │
│   Tal sistema não pode provar sua própria consistência.                     │
│                                                                             │
│   IMPLICAÇÕES:                                                              │
│   • Completude impossível (se consistente)                                  │
│   • Auto-verificação impossível                                             │
│   • O sonho de Hilbert: matematicamente impossível                          │
│                                                                             │
│   ─────────────────────────────────────────────────────────────────────     │
│                                                                             │
│   A RECEPÇÃO:                                                               │
│                                                                             │
│   • Von Neumann entendeu imediatamente — cancelou próprio trabalho          │
│   • Hilbert: dizem que ficou chocado, depois aceitou                        │
│   • Muitos matemáticos: levaram anos para absorver                          │
│   • Alguns: nunca aceitaram completamente                                   │
│   • Hoje: resultado fundamental, inquestionável                             │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
