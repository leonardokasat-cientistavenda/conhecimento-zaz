## 4. Elementos Biográficos EXPANDIDOS

```

│                                                                             │
│   ALAN TURING — VIDA COMPLETA (COMPLEMENTO A D.1)                           │

│   ═══════════════════════════════════════════════════════════════════════   │

│   INFÂNCIA REVISITADA (1912-1926)                                           │
│                                                                             │
│   • Nasceu em Londres, 23 de junho de 1912                                  │
│   • Pais na Índia — administração colonial                                  │
│   • Criado por tutores na Inglaterra — solidão fundamental                  │
│   • Christopher Morcom: primeiro amor, morreu em 1930                       │
│   • A morte de Christopher moldou Turing para sempre                        │
│   • Pergunta que ficou: a mente sobrevive ao corpo?                         │
│                                                                             │

│                                                                             │
│   • 1931: Entra em King's College, Cambridge                                │
│   • Estudou matemática com alguns dos melhores                              │
│   • 1935: Eleito Fellow de King's — aos 22 anos                             │
│   • Problema que o obcecou: o Entscheidungsproblem de Hilbert               │
│   • "Existe procedimento mecânico para decidir qualquer proposição?"        │
│   • 1936: Resolve o problema — a resposta é NÃO                             │
│   • Para provar, inventou a MÁQUINA DE TURING                               │
│                                                                             │
│   A MÁQUINA (1936)                                                          │
│                                                                             │
│   • "On Computable Numbers, with an Application to the                      │
│     Entscheidungsproblem"                                                   │
│   • 24 anos. Artigo que fundou a ciência da computação.                     │
│   • Não era sobre construir máquinas — era sobre o que é COMPUTAR           │
│   • Definição abstrata, precisa, universal                                  │
│   • Provou: há números que não são computáveis                              │
│   • Provou: o problema da parada é indecidível                              │
│                                                                             │
│   PRINCETON (1936-1938)                                                     │
│                                                                             │
│   • Foi estudar com Alonzo Church (que provou o mesmo, diferente)           │
│   • Church tinha o cálculo lambda, Turing tinha a máquina                   │
│   • Provaram que são equivalentes (Tese Church-Turing)                      │
│   • PhD em Princeton, 1938                                                  │
│   • Conheceu von Neumann — ofereceram-lhe posição                           │
│   • Recusou. Voltou para a Inglaterra. Guerra vinha.                        │
│                                                                             │
│   BLETCHLEY PARK (1939-1945)                                                │
│                                                                             │
│   • Recrutado antes da guerra começar                                       │
│   • Quebrou a Enigma — máquina de criptografia nazista                      │
│   • Contribuição incalculável para vitória aliada                           │
│   • Construiu a BOMBE — máquina para quebrar códigos                        │
│   • Depois: trabalhou no Colossus (primeiro computador eletrônico)          │
│   • Tudo secreto — só revelado décadas depois                               │

│   PÓS-GUERRA (1945-1954)                                                    │
│                                                                             │
│   • 1945: National Physical Laboratory — projeto ACE                        │
│   • 1948: Manchester — um dos primeiros computadores funcionais             │
│   • 1950: "Computing Machinery and Intelligence" (coberto em D.1)           │
│   • 1951-1954: Trabalho em morfogênese (padrões biológicos)                 │
│   • 1952: Preso por "indecência grave" (homossexualidade)                   │
│   • Escolheu castração química em vez de prisão                             │
│   • 1954: Morte por cianeto — suicídio provável                             │
│                                                                             │
│   ═══════════════════════════════════════════════════════════════════════   │
```

## 5. Contexto Histórico-Intelectual

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   O ENTSCHEIDUNGSPROBLEM                                                    │
│                                                                             │
│   ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│   O QUE ERA:                                                                │
│                                                                             │
│   Hilbert perguntou (1928):                                                 │
│   "Existe um PROCEDIMENTO MECÂNICO (algoritmo) que, dada qualquer           │
│    proposição matemática, decide se ela é verdadeira ou falsa?"             │
│                                                                             │
│   Se existisse: matemática seria completamente automatizável.               │
│   Máquinas poderiam fazer o trabalho de matemáticos.                        │
│                                                                             │
│   ─────────────────────────────────────────────────────────────────────     │
│                                                                             │
│   O PROBLEMA:                                                               │
│                                                                             │
│   Para responder, precisava definir "procedimento mecânico".                │
│   O que significa "algoritmo"? O que é "computar"?                          │
│                                                                             │
│   Duas respostas emergiram em 1936:                                         │
│                                                                             │
│   CHURCH (Princeton):                                                       │
│   • Cálculo lambda — funções abstratas                                      │
│   • Computável = expressável em cálculo lambda                              │
│                                                                             │
│   TURING (Cambridge):                                                       │
│   • Máquina de Turing — dispositivo abstrato                                │
│   • Computável = calculável por máquina de Turing                           │
│                                                                             │
│   Provaram que são EQUIVALENTES (Tese Church-Turing).                       │
│   E ambos provaram: NÃO existe tal procedimento universal.                  │
│   O Entscheidungsproblem tem resposta NEGATIVA.                             │
│                                                                             │
│   ─────────────────────────────────────────────────────────────────────     │
│                                                                             │
│   POR QUE TURING É ESPECIAL:                                                │
│                                                                             │
│   Church foi primeiro (por meses).                                          │
│   Mas Turing é mais lembrado. Por quê?                                      │
│                                                                             │
│   • A máquina de Turing é INTUITIVA                                         │
│   • Qualquer um pode entender: fita, cabeçote, regras                       │
│   • Conecta com a ideia de computador físico                                │
│   • Church era abstrato; Turing era visual                                  │
│   • A máquina UNIVERSAL foi insight adicional                               │
│   • Uma máquina que simula QUALQUER máquina                                 │
│                                                                             │
│   ─────────────────────────────────────────────────────────────────────     │
│                                                                             │
│   GÖDEL → TURING:                                                           │
│                                                                             │
│   Turing conhecia Gödel (1931).                                             │
│   O problema da parada é "primo" da incompletude.                           │
│   Ambos usam diagonalização/autorreferência.                                │
│                                                                             │
│   Gödel: Sistema não pode provar certas verdades sobre si.                  │
│   Turing: Máquina não pode decidir certas questões sobre si.                │
│                                                                             │
│   São limites irmãos. Estrutura similar, domínios diferentes.               │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
