## 5. Conceitos Técnicos

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   CONCEITOS CHAVE DE PRIGOGINE                                              │
│                                                                             │
│   ═══════════════════════════════════════════════════════════════════════   │
│                                                                             │
│   SEGUNDA LEI DA TERMODINÂMICA:                                             │
│                                                                             │
│   dS ≥ 0 (para sistema ISOLADO)                                             │
│   Entropia sempre aumenta ou permanece constante                            │
│   Implica: universo caminha para "morte térmica"                            │
│                                                                             │
│   ─────────────────────────────────────────────────────────────────────     │
│                                                                             │
│   SISTEMAS ABERTOS:                                                         │
│                                                                             │
│   dS = dS_interno + dS_troca                                                │
│   Podem ter dS < 0 localmente                                               │
│   Desde que exportem entropia para o ambiente                               │
│   Total do universo ainda aumenta                                           │
│                                                                             │
│   ─────────────────────────────────────────────────────────────────────     │
│                                                                             │
│   ESTRUTURAS DISSIPATIVAS:                                                  │
│                                                                             │
│   • Padrões ordenados que surgem longe do equilíbrio                        │
│   • Mantidos por fluxo constante de energia/matéria                         │
│   • Estáveis enquanto houver fluxo                                          │
│   • Exemplos: células de Bénard, tornados, células vivas                    │
│                                                                             │
│   ─────────────────────────────────────────────────────────────────────     │
│                                                                             │
│   BIFURCAÇÕES:                                                              │
│                                                                             │
│   • Pontos de instabilidade                                                 │
│   • Sistema pode ir para múltiplos estados                                  │
│   • Pequenas flutuações determinam caminho                                  │
│   • Introduzem NOVIDADE e IRREVERSIBILIDADE                                 │
│   • Determinismo quebra nesses pontos                                       │
│                                                                             │
│   ─────────────────────────────────────────────────────────────────────     │
│                                                                             │
│   SETA DO TEMPO:                                                            │
│                                                                             │
│   • Irreversibilidade não é ilusão — é fundamental                          │
│   • Vem da termodinâmica de não-equilíbrio                                  │
│   • "A irreversibilidade é a fonte de ordem"                                │
│   • Sem irreversibilidade, não haveria vida, história, novidade             │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```
