## 5. Conceitos Técnicos

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                                                                             │
│   CONCEITOS CHAVE DE HOLLAND                                                │
│                                                                             │

│                                                                             │
│   1. REPRESENTAÇÃO                                                          │

│      Múltiplas soluções candidatas                                          │

│                                                                             │

│      Função que avalia qualidade da solução                                 │
│      Melhores têm mais chance de reproduzir                                 │
│                                                                             │
│   4. SELEÇÃO                                                                │
│      Escolhe pais para próxima geração                                      │
│      Favorece os mais aptos                                                 │
│                                                                             │
│   5. CROSSOVER                                                              │
│      Combina genes de dois pais                                             │
│      Cria filhos com características de ambos                               │
│                                                                             │
│   6. MUTAÇÃO                                                                │
│      Pequenas mudanças aleatórias                                           │
│      Mantém diversidade                                                     │
│                                                                             │
│   ─────────────────────────────────────────────────────────────────────     │
│                                                                             │
│   SISTEMAS COMPLEXOS ADAPTATIVOS (CAS):                                     │
│                                                                             │
│   1. AGENTES                                                                │
│      Entidades que agem e reagem                                            │
│      Seguem regras locais                                                   │
│                                                                             │
│   2. AGREGAÇÃO                                                              │
│      Agentes formam grupos                                                  │
│      Comportamento coletivo emerge                                          │
│                                                                             │
│   3. FLUXOS                                                                 │
│      Recursos circulam pelo sistema                                         │
│      Redes de interação                                                     │
│                                                                             │
│   4. NÃO-LINEARIDADE                                                        │
│      Pequenas causas → grandes efeitos                                      │
│      Feedbacks amplificam ou amortecem                                      │
│                                                                             │
│   5. DIVERSIDADE                                                            │
│      Heterogeneidade é crucial                                              │
│      Sistemas homogêneos são frágeis                                        │

│      Agentes têm representações do ambiente                                 │

│                                                                             │

│      Componentes reusáveis                                                  │
```
