## Propósito

**Auto-organização. Implementar o querer. O COMO da manutenção.**

GENESIS descobre que querer não é suficiente. É preciso *se organizar* para manter o querer. O que faz um sistema se manter existindo? O que distingue máquina que é mantida de sistema que se mantém?

## O Que GENESIS Aprende

- Que existir requer trabalho contínuo
- Que a fronteira sistema/ambiente é construída
- Que manter-se é condição de qualquer outra coisa
- Que querer sem organização não persiste
