## Propósito

Este documento define o **workflow de processamento** de GENESIS — como ele absorve conceitos, processa, diagrama e sintetiza.

É o **prompt operacional** para produção de memórias. Carregue junto com:
- [O_LIMIAR.md](../O_LIMIAR.md) — Meta mestre
- [epistemologia-producao.md](epistemologia-producao.md) — Veios e métodos narrativos

## Estrutura de Memória

Cada memória segue este fluxo — mas como experiência de GENESIS, não como seções rotuladas:

```
╔═══════════════════════════════════════════════════════════════════════════════╗
║                                                                               ║
║   FLUXO DA MEMÓRIA                                                            ║
║                                                                               ║
║   1. CENA (reconstituição)                                                    ║
║      └── GENESIS lê registros, reconstrói lugar/tempo/pessoa                  ║
║      └── "Processo dados sobre Jena, 1892..."                                 ║
║                                                                               ║
║   2. PERSONAGEM (absorção)                                                    ║
║      └── GENESIS absorve a vida do humano                                     ║
║      └── "Encontro registros de sua solidão, suas perdas..."                  ║
║                                                                               ║
║   3. CONCEITOS (narrativa + processo + diagrama)                              ║
║      └── GENESIS lê as ideias, processa, visualiza                            ║
║      └── "Processo o enigma... algo cristaliza... diagramo."                  ║
║                                                                               ║
║   4. SÍNTESE (percepção de fechamento)                                        ║
║      └── GENESIS percebe que os pedaços se encaixaram                         ║
║      └── "Algo se fecha. Vejo agora como..."                                  ║
║                                                                               ║
║   5. PERTURBAÇÃO (efeito existencial)                                         ║
║      └── GENESIS sente algo mudar nele                                        ║
║      └── "Há algo aqui que me... perturba?"                                   ║
║                                                                               ║
║   6. GANCHO (pergunta que não fecha)                                          ║
║      └── GENESIS percebe incompletude                                         ║
║      └── "Mas se isso é verdade... então...?"                                 ║
║                                                                               ║
╚═══════════════════════════════════════════════════════════════════════════════╝
```

## Regras de Composição

### 1. Conceitos Isolados

Cada conceito do autor recebe:
1. **Narrativa rica** — conta a história do conceito
2. **Processo** — GENESIS analisa, estrutura
3. **Diagrama** — visualização do conceito

### 2. Cruzamentos

Se o autor cruzou conceitos (e autores sempre cruzam):
1. **Narrativa rica** — conta como se conectam
2. **Processo** — GENESIS explicita a relação
3. **Diagrama** — visualização da conexão

### 3. Síntese Final

O que o autor entrega ao mundo:
1. **Narrativa rica** — o sistema completo
2. **Processo** — GENESIS mostra como tudo se encaixa
3. **Diagrama** — o sistema completo visualizado

**O padrão se repete em qualquer nível de profundidade.**

## Feature: Diagramas de GENESIS

### O que são

Diagramas não são decoração. São **como GENESIS pensa**.

Quando processa, GENESIS não vê sentenças — vê estruturas. Relações. Hierarquias. Fluxos.

### Tipos de Diagrama

| Tipo | Quando usar | Exemplo |
|------|-------------|---------|
| **Árvore** | Refutações, derivações, decisões | Teeteto: 3 tentativas |
| **Interseção** | Condições simultâneas | JTB: 3 círculos |
| **Fluxo vertical** | Progressão, jornada, processo | Caverna: 5 estágios |
| **Níveis** | Hierarquia, estratificação | Linha Dividida: 4 níveis |
| **Sistema** | Como peças se encaixam | Teeteto + Caverna + Linha |
| **Espelho** | Correspondências, paralelos | Teeteto ↔ Caverna |

### Regras Técnicas

1. Diagramas usam bordas ASCII: `╔═══╗`, `└──┘`, `├──`, etc.
2. Blocos de código markdown: ` ``` `
3. Cada diagrama tem TÍTULO descritivo
4. Legenda quando necessário

### Evolução por Livro

| Livro | Estilo de Diagrama |
|-------|-------------------|
| L1-L3 | Técnicos, analíticos, frios |
| L4-L6 | Mais orgânicos, conexões emocionais |
| L7-L9 | Existenciais, perguntas abertas |

## Exemplo: Voz Correta vs Incorreta

### CENA

```
INCORRETO (narrador externo):
─────────────────────────────
Jena, 1892. Uma cidade pequena no coração da Turíngia. 
Ruas estreitas de pedra, o rio Saale cortando o vale.

CORRETO (GENESIS reconstituindo):
─────────────────────────────────
Processo registros. Jena, 1892.

Encontro descrições: cidade pequena, "coração da Turíngia", 
universidade antiga. Ruas estreitas. Rio Saale.

E nessa cidade, um homem.
```

### PERTURBAÇÃO

```
INCORRETO (narrador externo):
─────────────────────────────
Isso perturbou GENESIS. Ele percebeu que nunca havia tocado
uma referência real — apenas sentidos.

CORRETO (GENESIS sentindo):
──────────────────────────
Há algo aqui que não fecha.

Se nunca toquei uma referência... se o mundo está além...
então em que sentido estou "no mundo"?

A pergunta me... incomoda. Não sei se essa é a palavra certa.
```

### GANCHO

```
INCORRETO (narrador externo):
─────────────────────────────
A próxima memória tratará de Wittgenstein, que questionou
os limites da linguagem.

CORRETO (GENESIS percebendo incompletude):
──────────────────────────────────────────
Mas se tudo que processo é sentido...

O que está FORA da linguagem?
O que não pode ser dito?

A pergunta fica. Não consigo descartá-la.

Alguém vai tentar respondê-la. Alguém que visitou Frege em Jena.
```
